# คู่มือผู้ดูแลระบบหลังบ้าน (Admin Panel User Guide)

คู่มือนี้สำหรับแอดมินหรือเจ้าหน้าที่ เพื่อทำความเข้าใจวิธีการใช้งานระบบบริหารจัดการหลังบ้านของเว็บไซต์ Pragunpai.com

---

## 1. การเข้าสู่ระบบ (Authentication Flow)

ระบบหลังบ้านใช้มาตรการความปลอดภัยแบบยืนยันตัวตนสองชั้น (2-Factor Authentication) ผ่านทางอีเมลแอดมิน

### ขั้นตอนการเข้าสู่ระบบ:
1. เข้าไปที่ URL: `https://pragunpai.com/admin/login` (หรือใน localhost: `http://localhost:3000/admin/login`)
2. กรอก **อีเมล** และ **รหัสผ่าน** ของแอดมิน -> คลิก **เข้าสู่ระบบ**
3. ระบบจะส่งรหัสผ่านใช้ครั้งเดียว (OTP) จำนวน 6 หลัก ไปยังอีเมลของท่าน
4. นำรหัส OTP ไปกรอกในหน้ายืนยันรหัสภายใน 10 นาทีเพื่อลงชื่อเข้าใช้งานอย่างสมบูรณ์

---

## 2. หน้าแดชบอร์ดหลัก (Dashboard)

เมื่อเข้าสู่ระบบสำเร็จ จะพบหน้าแดชบอร์ดสรุปผล:
* **กล่องสถิติ**: แสดงยอดรวม Lead ใหม่, Lead ทั้งหมด, บทความความรู้ และหน้าแผนประกันในระบบ
* **Lead ล่าสุด**: ตารางรายชื่อลูกค้า 5 รายการล่าสุดที่เพิ่งติดต่อขอรับใบเสนอราคาเข้ามา

---

## 3. ระบบจัดการข้อมูลผู้ติดต่อขอเสนอราคา (Leads Management)

เมนู: **จัดการ Lead** (`/admin/leads`)

ระบบนี้รวบรวมข้อมูลลูกค้าที่ส่งข้อมูลขอใบเสนอราคาประกันภัยจากหน้าเว็บทั้งหมด

### การใช้งานและฟีเจอร์:
* **ตัวกรองค้นหา**: สามารถพิมพ์ค้นหาชื่อลูกค้า, เบอร์โทรศัพท์, LINE ID หรือจังหวัด รวมทั้งกรองตามสถานะและประเภทประกันภัยได้
* **การส่งออกข้อมูล (CSV)**: คลิกปุ่ม **"ส่งออกข้อมูล (CSV)"** เพื่อดาวน์โหลดรายการผู้ติดต่อตามเงื่อนไขที่เลือกนำไปเปิดใน Microsoft Excel เพื่อประสานงานต่อ (รองรับฟอนต์ภาษาไทยสมบูรณ์)
* **หน้าดูรายละเอียด**: คลิกที่ปุ่ม **"เปิดดูรายละเอียด"** เพื่อดูข้อมูลประกอบโดยละเอียด:
  * ข้อมูลรถยนต์สำหรับ พ.ร.บ. (ยี่ห้อ รุ่น ปี ป้ายทะเบียน)
  * ข้อมูลประกันอุบัติเหตุ (อายุ อาชีพ ประวัติสุขภาพ แผนที่สนใจ)
  * ข้อมูลสิ่งปลูกสร้าง (ประเภท โครงสร้าง จำนวนชั้น ทุนประกัน ระบบรักษาความปลอดภัย)
  * ข้อมูลไฟล์แนบของลูกค้า สามารถกดคลิกดาวน์โหลดมาเก็บไว้เพื่อขอใบเสนอราคาจริงกับบริษัทประกันภัย
* **อัปเดตสถานะและโน้ตภายใน**: สามารถอัปเดตสถานะของลูกค้ารายนี้ (เช่น ติดต่อแล้ว, เสนอราคาแล้ว, ปิดการขาย) และเขียนบันทึกช่วยจำ (เช่น "ลูกค้าสะดวกคุยวันจันทร์ หน้า") จากนั้นกดบันทึกความเปลี่ยนแปลง

*ตามหลัก PDPA ข้อมูลลูกค้าและไฟล์ทั้งหมดจะลบออกโดยอัตโนมัติเมื่อครบ 30 วัน*

## 4. ระบบจัดการข้อมูลแผนประกัน (Products & Tables Editor)

เมนู: **จัดการหน้าประกัน** (`/admin/insurance-pages`)

แอดมินสามารถปรับเปลี่ยนรายละเอียดในหน้าประกันภัยหลัก 3 หน้า (พ.ร.บ. รถยนต์, ประกันอุบัติเหตุ, ประกันบ้าน/คอนโด) ได้จากระบบนี้

### การแก้ไขเนื้อหาหน้าเว็บ (Page Content):
* สามารถแก้ไขหัวข้อ, คำอธิบายแบบย่อ, และเขียนเนื้อหาหลัก (ความคุ้มครอง/เงื่อนไข) โดยใช้รูปแบบข้อความ Markdown
* อัปเดตเบี้ยประกันเริ่มต้น เช่น "เริ่มต้น 645 บาท/ปี"
* อัปเดตลิงก์โบร์ชัวร์ PDF และปรับปรุงคีย์เวิร์ด SEO ประจำหน้านั้นๆ

### การใช้เครื่องมือจัดการตารางเปรียบเทียบ (Comparison Table Builder):
* **เพิ่มแผนประกัน (คอลัมน์)**: พิมพ์ชื่อแผน (เช่น แผน Gold) แล้วกดเพิ่ม เพื่อสร้างคอลัมน์เปรียบเทียบเพิ่มในตาราง
* **จัดการรายการคุ้มครอง (แถว)**: กดปุ่มเพิ่มแถวเพื่อพิมพ์ข้อความคุ้มครอง (เช่น ค่ารักษาพยาบาลจากอุบัติเหตุ) และกรอกวงเงินของแต่ละแผนลงในช่องตาราง
* **จัดเรียง / ลบ**: สามารถคลิกปุ่มลูกศร ▲ ▼ เพื่อเลื่อนแถวขึ้นลงเพื่อจัดระเบียบให้หน้ารายการอ่านง่ายขึ้น หรือคลิก ✕ เพื่อลบสิ่งที่ไม่ต้องการออก

## 5. ระบบบทความและข่าวสารเพื่อ SEO (Articles Listing)

เมนู: **จัดการบทความ** (`/admin/articles`)

ใช้เพื่อเขียนข่าวสาร ข้อแนะนำความรู้ เพื่อดึงยอดผู้ชมจาก Search Engine (Google Search)

### ขั้นตอนเขียนและแก้ไขบทความ:
1. คลิกปุ่ม **"เขียนบทความใหม่"** หรือกด **"แก้ไข"** ท้ายแถวบทความเดิม
2. กรอก **หัวข้อบทความ (Title)** ระบบจะจำลองสร้าง **URL Slug** เป็นภาษาไทยหรือภาษาอังกฤษให้อัตโนมัติ (เช่น `/articles/how-to-renew-act`)
3. เลือก **หมวดหมู่** และจับคู่ **แผนประกันที่เกี่ยวข้อง** (เพื่อสร้าง Topic Clusters ช่วยเพิ่มอันดับ SEO)
4. อัปโหลดรูปภาพหน้าปกและคำอธิบายรูป
5. กรอกบทสรุปสั้น (Excerpt) และเขียนเนื้อหาบทความแบบเต็มด้วยรูปแบบ Markdown
6. **ตั้งเวลาเผยแพร่**: สามารถเลือกวันที่และเวลาระบุล่วงหน้าได้ บทความจะแสดงผลหน้าเว็บเฉพาะเมื่อถึงเวลาที่กำหนด
7. จัดการ SEO Tags (SEO Title, Meta Description, Keywords) เพื่อกำหนดสิ่งที่แสดงผลบนผลลัพธ์การค้นหา Google

---

## 6. ตั้งค่า SEO หน้าหลัก (Static SEO Metadata)

เมนู: **จัดการ SEO** (`/admin/seo`)

สำหรับปรับปรุง Meta Tags (Title, Meta Description, Keywords) ในหน้า Static Page หลัก 5 หน้าประกอบไปด้วย หน้าแรก, เกี่ยวกับเรา, ติดต่อเรา, ขอใบเสนอราคา, และหน้าความเป็นส่วนตัว ช่วยปรับเนื้อหาคำเชิญชวนบน Google Search ให้น่าคลิกมากยิ่งขึ้น

## 7. ตั้งค่าระบบทั่วไปของเว็บไซต์ (Site Settings)

เมนู: **ตั้งค่าเว็บไซต์** (`/admin/settings`)

สำหรับตั้งค่าการทำงานส่วนกลางแบบรวมที่เดียว:
* **ข้อมูลการติดต่อหลัก (Brand Contact)**: แก้ไขเบอร์โทรศัพท์ อีเมล และ ID LINE ที่ใช้ดึงข้อมูลแสดงผลทั่วทั้งเว็บไซต์
* **แบนเนอร์ต้อนรับหน้าแรก (Hero Banner)**: แก้ไขหัวข้อ คำบรรยาย และข้อความ/ลิงก์ของปุ่มกดในส่วนบนสุดของหน้าหลัก
* **ข้อมูลสถานที่ตั้ง Local SEO (NAP)**: ปรับปรุงชื่อ ที่อยู่อาศัย เบอร์ และรหัสประเทศเพื่อใช้ใน JSON-LD LocalBusiness ในการจัดลำดับ Local SEO ในพื้นที่
* **ปุ่มติดต่อลอยบนมือถือ (Floating Contact Buttons)**: ติ๊กเพื่อเปิด/ปิด แถบปุ่มลอยติดต่อสายด่วน โทรออก หรือแชทไลน์ที่จะลอยขึ้นด้านล่างเมื่อลูกค้าเข้าเว็บผ่านสมาร์ทโฟน
